Active Developer Badge

Tutorial by conicey

  • Go to https://discord.com/developers/applications, and create a new application on the account you want the badge on.
  • Go to the Bot tab and generate a bot account for the application.
  • Click the “Reset Token” button to generate a token (if you have 2FA enabled, you'll need your code).
  • Copy the token to your clipboard.
  • Click this badge to clone this repo to a new repl in Replit → Run on Repl.it.
  • You will be redirected to Replit — you may need to create an account (you can register with Google).
  • Replit runs online — it does not run on your computer.
  • Wait for the repo to finish cloning and the Repl to connect fully.
  • Click the green “Run” button at the top.
  • Go to the console and paste in the token you copied earlier. (Note: CTRL + V won’t work, use CTRL + SHIFT + V or right-click → paste.)
  • Press the Enter key.
  • Wait for it to print “DONE | Application/Bot is up and running”. Do not close the tab or stop the program — the bot will go offline!
  • Go back to your application page → General Information tab → scroll down and copy the Application ID.
  • Create a new Discord server (you can delete it later).
  • Enable Community on the server (if you don’t already own a community-enabled one).
  • Return to https://discord.com/developers/applications.
  • Copy the Application ID.
  • Use this invite link to add the bot to the server:
    https://discord.com/oauth2/authorize?client_id={applicationid}&scope=bot%20applications.commands&permissions=105227086912
  • In the server, use the /ping command on your bot.
  • Go to the Active Developer page and register your bot.
  • If it says you’re not eligible, wait up to 24 hours — it’s just syncing with Discord’s system.
NOTE: Make sure you have the “Use data to improve Discord” setting enabled under User Settings → Privacy & Safety, otherwise you won’t be eligible.
NOTE 2: You don’t need to keep the Replit tab open 24/7 — only open it when you want to run your bot.
